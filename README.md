Personal Finance Dashboard | Power BI
📌 Project Overview

This project is an interactive Personal Finance Dashboard built using Power BI to analyze income, expenses, savings, investments, and overall financial performance over time. The dashboard transforms raw financial records into meaningful insights, helping users monitor their financial health, identify spending patterns, and make data-driven financial decisions.

The project demonstrates skills in Power BI, Power Query, Data Modeling, DAX, Data Cleaning, and Dashboard Design.

🎯 Business Problem

Managing personal finances through spreadsheets or manual records makes it difficult to answer important financial questions such as:

Is monthly income sufficient to cover expenses?
Which expense categories consume the highest portion of income?
How much money is being saved each month?
Are investments increasing over time?
What are the spending trends across different months and years?
Which financial components require immediate attention?

Without proper visualization and analysis, individuals struggle to track cash flow, control unnecessary expenses, and achieve financial goals.

💡 Solution

This Power BI dashboard provides a centralized view of financial data by:

Tracking total income, expenses, savings, and investments.
Comparing monthly financial performance.
Identifying high-spending categories.
Monitoring savings growth over time.
Visualizing financial trends using interactive charts.
Providing KPIs for quick financial assessment.

The dashboard enables users to make informed financial decisions based on real-time insights.
